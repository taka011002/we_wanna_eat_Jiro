# Cheers!

知らない人とオンライン飲み会を気軽にできるサービス「Cheers!」です!

URL -> https://cheers.reud.net/

## 機能

- Googleログイン(Firebase Authentication)
- 4人までのテレビ通話(SFUを利用)
  - テレビ通話画面でユーザ表示名を自動表示
  - 他のユーザのルームへの入退出に応じて自動画面描画
- チャット用ルーム作成
- ユーザ表示名変更(ルームに入るとき, 部屋を作成するとき)

探せば他にも機能はありそう...

## 使用技術

- Frontend
  - Next.js(SSR)
  - TypeScript
  - Material-UI
  - Skyway
- Backend
  -  Firebase
  - Authentication
  - Firestore
